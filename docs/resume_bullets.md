# Resume Bullets

- Built a Databricks-based product analytics lakehouse with bronze/silver/gold Delta tables, idempotent `MERGE` processing, and analytics-ready marts for activation, adoption, retention, and experimentation.
- Implemented reusable Python modules plus Databricks notebooks for schema enforcement, data quality validation, audit logging, and operational runbooks, mirroring production-style engineering standards.
- Developed an LLM-style feedback enrichment workflow that classified support tickets into product topics and sentiment labels, integrating structured and unstructured data into curated analytics outputs.
- Modeled incremental batches, late-arriving records, duplicate events, and contract violations to show rerun safety, quarantine handling, and trustworthy metric publishing.
